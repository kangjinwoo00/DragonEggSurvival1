# 🐉 Dragon Egg Survival

마인크래프트 Paper 서버용 커스텀 콘텐츠 플러그인  
드래곤 알을 둘러싼 100일간의 생존 경쟁 서버

---

## 📌 서버 핵심 규칙

- 마인크래프트 시간 기준 **99일 진행**
- **100일차에 드래곤 알 보유자 즉시 우승**
- 엔딩 없음 / 외부 엔드섬 이동 불가

---

## 🥚 드래곤 알 시스템

- 우클릭 시 흡수
- 보유자에게 강력한 페널티 부여
  - 발자국 추적 (낮 / 5일)
  - 엔더 이펙트 + 사운드
  - 차원 이동 제한
- 사망 시 제단 + 드래곤 알 생성
- 보라색 신호기 빛 100분 유지

---

## 🚫 제한 사항

- 채팅 / 좌표 / 발전과제 메시지 차단
- 주민 거래 하루 20회
- 포션 최대 2개
- 황금사과 재생 너프
- 금지 아이템:
  - 겉날개 / 엔더 진주 / 셜커 상자 / 방패
  - 불사의 토템 / 삼지창 / 엔더 상자
- 인챈트 제한:
  - 날카로움 최대 10
  - 보호 최대 7
  - 수선 / 무한 금지

---

## 🌍 월드 설정

| 차원 | 크기 |
|----|----|
| 오버월드 | 1000 × 1000 |
| 네더 | 500 × 500 |
| 엔드 | 400 × 400 |

---
